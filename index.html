Backup dos dados

Este app já salva tudo automaticamente no seu celular (nesse navegador). Ainda assim, faça backups de tempos em tempos — se você limpar os dados do navegador, desinstalar o app do GitHub Pages da tela inicial ou trocar de celular, os dados salvos aqui se perdem. O backup é sua cópia de segurança fora do navegador.

Isso substitui os dados atuais do app pelos dados do arquivo escolhido. Use se perder os dados ou trocar de celular.
